How to use CSS Viewer

Tutorial:

1. Install and Open

- After installing the extension. You click on the icon of that extended version.

Please Click Here To Vote For Us

2. Start Using

- To start using the extension, please click on the Start button first.



- To view the css elements of a web page, move the cursor to the selection that you want to check.

- Click the left mouse button on the outline of the selected area. Then press F key on your keyboard to freeze popup on that web page. (if you don't want it to move around)





- If you want to copy all the CSS, press C, it will bring up a prompt, press Ctrl + C to copy to the clipboard, and you can paste them anywhere you want to edit ...
